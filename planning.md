# friendship-match
A friend-matching platform for university students. Uses interest-based profiles, compatibility scoring, and matching algorithms to help make social connections. Designed for the Friends Across Campus club.

## Features
- MVP
    - App
        - Has main match algorithm
    - Admin
        - Can make and edit compatibility questions (multiple coice, scale, true/false)
        - Can match people up with match algorithm
    - Friend finder
        - Can fill out questions
        - Can get results via email
- More features
    - App
        - Algorithm is customizable to every "Friend Finder" by their chosen importance (weight) for every category
        - Base algorithm adjusts itself according to the avg/median/mode weights of categories
    - Admin
    - Friend finder
        - Can check compatibility by category
        - Can change importance (weight) of category
        - Can save progress and finish questions later
        - Can save previous answers to apply to new friendship matches
        - Can rate match (good or bad)
